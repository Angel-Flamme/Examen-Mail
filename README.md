# Examen-Mail
Ceci est une solution 100% locale (sur le PC et non pas sur internet) pour apprendre aux usagers découvrant l'informatique dans une association, comment envoyer et lire un mail. Ce projet n'est pas réalisé pour être une solution complète (réponse, suppr...), sauf si je reçois une demande de mon supérieur dans quel cas je mettrais à jour le poste.
